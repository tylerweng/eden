# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users/:userName`
- `PATCH /api/users/:userName`

### Artist

- `GET /api/:artistName`

### Album

- `GET /api/:artistName/:albumTitle`

### Track

- `GET /api/:artistName/:albumTitle/:trackTitle`
